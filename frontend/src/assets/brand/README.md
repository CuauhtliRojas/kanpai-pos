# Assets de marca

Esta carpeta contiene los archivos visuales reemplazables de Kanpai POS.

- `app-logo.svg`: logo horizontal para topbar y menús.
- `app-icon.svg`: marca compacta para espacios reducidos.
- `product-placeholder.svg`: imagen neutra para productos sin fotografía.

Para personalizar la aplicación, reemplaza el archivo correspondiente conservando exactamente su nombre y extensión. No es necesario cambiar imports, rutas ni componentes.

Mantén los SVG simples, locales y sin scripts, recursos remotos ni datos incrustados de gran tamaño. Optimiza las imágenes antes de incorporarlas para no aumentar el tiempo de carga ni competir visualmente con la operación.
